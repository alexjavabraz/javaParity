# javaParity

[![B|Solid](http://findicons.com/files/icons/2155/social_media_bookmark/32/blogger.png)](http://blogjavabraz.blogspot.com.br/2017/07/segundo-tutorial-java-e-etereum-smart.html) Acesse o tutorial no Blog.

Este projeto é um tutorial em Java para iteração com a rede Blockchain Ethereum, utilizando:
  - Java 1.8
  - Maven
  - Parity

# Java e Ethereum

  - Crie um Smart Contract
  - Compile com o Solc
  - Gere os wrappers Java com o web3J
  - Faça deploy do Contrato pelo Parity
  - Faça a chamada do contrato dentro do programa Java


### Installation

Você precisará ter o JDK instalado [JDK 1.8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html) .

Você precisará ter também o [Maven](https://maven.apache.org/download.cgi) instalado.

Baixando o projeto do GIT.

```sh
$ git clone https://github.com/alexjavabraz/javaParity.git
$ cd javaParity
$ mvn clean install
```
